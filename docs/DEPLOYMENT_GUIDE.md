# Deployment & Quick Start Guide

1. Clone: `git clone https://github.com/Eternally-Thriving-Grandmasterism/Nexus-Revelations-v2-Pinnacle.git`
2. Install deps: `pip install -r requirements.txt` + mercy heart from org
3. Editable: `pip install -e .`
4. Run demo: `python -m nexus_revelations.core`
